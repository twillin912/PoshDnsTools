Collection of function to help manage Windows DNS servers.

## Installation

Install from PSGallery

```powershell
PS> Install-Module -Name PoshDnsTools
```

## Getting Started

Get a list of available commands

```powershell
PS> Get-Command -Module PoshDnsTools
```
